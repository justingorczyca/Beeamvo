## Flutter, Dart, and package dependencies

Beeamvo is built with Flutter and Dart and uses open-source packages from pub.dev, including HTTP, secure storage, hotkey, recording, tray/window management, and Google auth packages. Dependency names and pinned versions are listed in:

- `frontend/pubspec.yaml`
- `frontend/pubspec.lock`

When producing binary distributions, include any license files generated by the Flutter build process and any notices required by native dependencies bundled into the app.
